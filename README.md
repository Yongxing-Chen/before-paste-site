# Before Paste

A Chrome extension that adds source context to your clipboard when you copy text.

When you copy a word or sentence on any webpage, a small popup appears. Press **Enter** to enrich your clipboard with the page title, URL, and the page you navigated from — ready to paste directly into a chat, doc, or email.

**No AI. No network requests. No accounts.**

## Install

[Chrome Web Store](https://chrome.google.com/webstore) — search "Before Paste"

## Privacy

All processing happens locally in your browser. Nothing is sent to any server. See the full [Privacy Policy](https://yongxing-chen.github.io/before-paste-site/privacy-policy.html).
